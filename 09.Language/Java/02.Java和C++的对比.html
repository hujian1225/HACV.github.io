                # 01.可以将『布尔值』与『整数』进行比较吗 ？

                • Java不可以！
                • C和C++可以

                关键字区别：

                • Java中布尔值是boolean

                • C99及以后和C++中是bool

                菜鸟教程，C 语言的布尔类型(true 与 false)

                • C 语言标准(C89) 没有定义布尔类型，如果你使用 true 和 false
                • C语言并非没有bool类型，而是在C99标准之前没有
                  • 『C99标准添加了_BOOl类型，用于表示布尔值』
                  • C语言用值1表示true，值0表示false
                  • 参考自：《C Primer Plus》P54
                /*
 * ISO C Standard:  7.16  Boolean type and values  <stdbool.h>
 */
#ifndef _STDBOOL_H
#define _STDBOOL_H

#ifndef __cplusplus

#define bool	_Bool	//『『看这里』』
#define true	1		//『『看这里』』
#define false	0		//『『看这里』』

#else /* __cplusplus */
/* Supporting <stdbool.h> in C++ is a GCC extension.  */
#define _Bool	bool
#define bool	bool
#define false	false
#define true	true

#endif /* __cplusplus */

/* Signal that all the definitions are present.  */
#define __bool_true_false_are_defined	1

#endif	/* stdbool.h */

                # 02.内嵌基本数据类型的范围固定吗？

                • Java中内嵌基本数据类型的范围是固定的
                • C和C++中内嵌基本数据类型的范围是依照『具体实现』

                # 03.内存泄漏

                • 内存泄漏是不仅仅出现在C/C++程序的问题
                • Java程序也会内存泄漏！
                  • java是自动管理内存的，但是也会有内存泄露，比如加入HashMap的对象hash值改变了就无法从HashMap中remove，这就造成了内存泄露

                # 04.反射

                • C和C++ 本身并不支持反射
                • Java 支持反射机制

                访问private

                类中的数据域使用private修饰为私有变量 反射也可以访问 什么是反射访问？？？

                • 通过类的层面，来取得类中方法

                # 05.静态类

                • C#和Java中有静态类
                • C和C++中没有静态类

                # 06.接口类、抽象类、聚合类

                • 接口类：仅仅含有含有pure virtual （纯虚函数）的类
                • 抽象类：含有pure virtual （纯虚函数）的类

                So，抽象类和接口都不能被实例化

                类不可以多继承，而接口可以多实现
