          # 常用电脑运行命令集锦

          • 『命令行』运行重要命令『记忆』

          运行命令的意义：

           『命令』在所有Windows操作系统上统一

          • 快捷键缺点：会有不同公司生产的机器不同的区别
          • 比如『文件改名』，ASUS可以直接用F2，但是联想小新，需要Fn+F2
          • 方案一、一般的运行命令我们按下『win+R』在『运行』窗口中输入cmd再进行操作
          • 方案二、『注意』Control Panel 控制面板，或许cpl是这个后缀来源？ 然后，凡是cpl的，似乎都需要打全.cpl，不允许省略后缀！！！
          • PS：最好的打开所有『控制面板功能』的命令control——打开控制面板
          • 方案三、（请全部用本方案）某些命令，不能在cmd中使用，而是直接在我们按下『win+R』在『运行』窗口中输入

          记忆电脑运行命令的理由：

          相较于『快捷键』的遍历，似乎这种方式不是太好，但是当你遇到在“联想”电脑上重命名快捷键是Fn+F2，但是在ASUS电脑上是F2的时候，你就会觉得麻烦。因为不统一

          命令行的优势在于，『统一』全部是这样的命令

          • 最重要的几个管理：控制面板、注册表管理、环境变量

          # （2）『控制面板类』

          注意：必须写后缀cpl

          参考常见控制面板命令

          • sysdm.cpl——打开系统熟悉，比如可以去设置“环境变量”『重要』

            • 场景：Java配置3个环境变量、自定义自己的『运行命令』

          • intl.cpl 进行“区域”的设置，包括整个系统的编码是Unicode吗，中文还是英语，

            • 场景：由于系统编码是中文，会使得我们的Windows的『SRIM』软件有问题，打不开，是区域和语言选项（输入法选项）

          • ncpa.cpl 网络连接

            • 比如，我的无线等网络各种的设置、『禁用某块网卡』

          • appwiz.cpl 打开那个“卸载或更改程序”，要是除了Windows10之外的系统，不能快捷键win+X+F就能用这个补全

          • firewall.cpl 设置WINDOWS防火墙

          • desk.cpl 显示器属性（屏幕分辨率）

            • //用于修复，学校『投影幻灯片』等问题

          • mmsys.cpl----声音和音频设备 （设置声音，那些什么立体，也就是录制的是麦还是电脑本身）

            • 比如：QQ共享屏幕，声音的输入，输出
            • PS：这个关闭，可以有效关闭，某些软件可能对你的背后的『监听行为』

          # （3）『注册表类』

          • regedit--------注册表（编辑器）

          • regedt32-------注册表编辑器（同上）

            • window操作系统管理，难点所在，比如『mysql删不清』、『鼠标右键的那些应用设置』

          • lusrmgr.msc----本机用户和组 （win10可以不用这个来管理）

          • gpedit.msc-----组策略（为啥打不开？打开之前，需要设置，其实是能打开的）

          # （4）『微软管理控制台文件』

          注意：必须写后缀msc (Microsoft Management Control)

          • services.msc——本地服务设置
            • 替代快捷键之后，去找『服务』的步骤
          • diskmgmt.msc---磁盘管理实用程序
            • 比如，可以查看磁盘哪个分区是NTFS格式啥的
          • chkdsk.exe-----Chkdsk磁盘检查 （可能要谨慎使用）
            • 或许可以用于、我装PE盘的时候进行『格式化』
